# Imagens do E-mail JPR 2026

Esta pasta deve conter as imagens usadas no template de e-mail dos convites JPR 2026.

## Arquivos necessários:

### Banners principais:
- `Banner_JPR26.png` - Banner principal do evento JPR 2026
- `Banner_RX570.png` - Banner do Monitor Médico Eizo RX570MD
- `banner_Medecom3.png` - Banner da Workstation MEDECOM (Lumix)
- `banner_logo_RADIX_EIZO.png` - Logo RADIX Eizo

### Imagens de instrução:
- `Imagem_inst_1.png` - Instrução de como fazer inscrição no sistema
- `Imagem_inst_2.png` - Instrução de como inserir o código do convite

## Importante:

As imagens devem estar nesta pasta para que sejam servidas corretamente pela aplicação. Elas serão acessíveis via URL `/image/NomeDoArquivo.png`.

**Status atual:** As imagens foram enviadas via interface do AI Studio, mas não estão commitadas no Git. Para garantir persistência e versionamento, faça upload dos arquivos de imagem diretamente nesta pasta via Git.

## Como adicionar as imagens:

1. Vá para esta pasta no repositório: `public/image/`
2. Clique em "Add file" > "Upload files"
3. Faça upload de todos os 6 arquivos PNG listados acima
4. Commit as alterações

Isso garantirá que as imagens estarão disponíveis mesmo após redeployment da aplicação.
